Plugin documentation

Goals available:

Goal Description
glassfish:stop-domain Stop a domain which is currently running in a local or remote Glassfish instance
glassfish:redeploy "Hot" redeploy JavaEE components to a domain in a local or remote Glassfish instance
glassfish:undeploy Undeploy JavaEE components which are currently deployed to a domain in a local or remote Glassfish instance
glassfish:deploy Deploy JavaEE component artifacts to domain in a local or remote Glassfish instance
glassfish:delete-domain Delete an existing domain in a local or remote Glassfish instance
glassfish:create-virtual-server Create a new virtual server in a local or remote Glassfish instance
glassfish:start-domain Start an existing domain in a local or remote Glassfish instance
glassfish:create-domain Create a new domain in a local or remote Glassfish instance
